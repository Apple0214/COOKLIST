# COOKLIST

## サイト概要
料理レシピ投稿サイト

### サイトテーマ
料理を作るため、個人のレシピの公開

### テーマを選んだ理由
自炊することになり、レシピを確認する機会が増えて料理に関して興味がわいたため。

### ターゲットユーザ
料理に興味のある方全般に対応 幅広く

### 主な利用シーン
一つの料理に対して、たくさんのレシピを知ることができて、お気に入りのものを見つけられる。
自分のレシピを公開し、みんなに教えられる。
いろいろなレシピを見ることで自分のレシピの選択肢が増える
